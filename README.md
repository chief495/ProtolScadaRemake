# ProtolScadaRemake

![Protol Logo](Images/ProtolLogo.ico)

Система диспетчерского управления технологическим процессом для производства буровзрывных работ.

## 📋 Описание проекта

SCADA-система для мониторинга и управления технологическими процессами на производстве Протол. Проект представляет собой модернизированную версию существующей системы с улучшенным интерфейсом и функциональностью.

## 🚀 Основные возможности

- **Мониторинг в реальном времени** технологических параметров
- **Управление производственными линиями**:
  - ГГД (Газогенераторные устройства)
  - ГРО (Гидравлические разгрузочные устройства)
  - ТС (Транспортные системы)
  - ЭМ (Электромеханическое оборудование)
- **Рецептурное управление** производственными процессами
- **Аварийная сигнализация** и ведение журнала событий
- **Исторические данные** и отчетность

## 🖥️ Технологический стек

- **Платформа**: .NET Framework / .NET Core
- **Язык**: C#
- **UI Framework**: WPF (Windows Presentation Foundation)
- **Архитектура**: MVVM (Model-View-ViewModel)
- **Сборка данных**: OPC UA / Modbus (если используется)
- **База данных**: MySQL
